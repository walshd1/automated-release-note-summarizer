# Automated Release Note Summarizer

Summarizes commit messages between releases to automatically generate concise and informative release notes.

## Free
```yaml
- uses: walshd1/automated-release-note-summarizer@v1
  with:
    gemini_api_key: ${{ secrets.GEMINI_API_KEY }}
```

## Paid (cost + 4.75%)
```yaml
- uses: walshd1/automated-release-note-summarizer@v1
  with:
    service_token: ${{ secrets.ACTION_FACTORY_TOKEN }}
```
